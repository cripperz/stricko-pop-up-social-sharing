== Description ==
Stricko pop-up social sharing plugin allow you to increase your traffic by pop-up jquery box containing facebook like button , twitte , google + , Linkedin



== Installation ==
1. Go to the wp-admin/ section of your website
2. Select the Plugins menu item
3. Select the Add New sub-menu item
4. Select upload and choose your file .zip
5. Click activate plugin ... here is it !
